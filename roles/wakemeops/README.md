# WakeMeOps

This role installs and configures the WakeMeOps repository, providing access to a collection of modern packages for
development environments.

## Requirements

- Debian-based Linux distribution (Ubuntu, Debian)
- Ansible 2.1+

## Role Variables

No variables.

## Dependencies

No dependencies.

## Example Playbook

```yaml
- hosts: localhost
  roles:
    - wakemeops
```
